### Project Description

A modern **JavaFX GUI Temperature Converter** that allows users to convert temperature values between **Celsius**, **Fahrenheit**, and **Kelvin** with a clean and user-friendly interface.

### Features

- Clean and responsive graphical user interface
- Support for Celsius, Fahrenheit, and Kelvin conversions
- **Swap Units** button for easy conversion in both directions
- Input validation with proper error messages
- Exception handling and assertions
- Modular design with well-documented methods
- Modern styling using CSS (rounded components, hover effects, professional look)
- Full Javadoc documentation

### Technologies Used

- Java
- JavaFX
- CSS for styling

### How to Run

1. Make sure the `lib` folder (JavaFX SDK) is in the project root
2. Double-click `run.bat` (Recommended to run as Administrator)
